String Methods

Strings are immutable.

1. str.trim() - to remove starting and ending spaces.

2. str.toUpperCase() - to make strings uppercase.

3. str.toLowerCase() - to make string lowercase.

4. str.indexOf() - to find the index of string's first occurance.

5. str.slice() - to cut the string.

6. Method Chaining - we can write string method one after another.

7. Replace Method - only first occurance is replaced in the string

8. Repeat Method - it repeats the same string by taking argument of how many time
